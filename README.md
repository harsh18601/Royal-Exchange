# Royal Ruby & Sapphire Exchange

A premium, AI-driven luxury gemstone commerce platform built with Next.js 14 and Contentful CMS.

## 💎 Features

- **Luxury Esthetics**: High-fidelity dark-themed UI with glassmorphism and premium animations.
- **Contentful CMS**: Dynamic gemstone catalog management via Contentful.
- **GemAI Advisor**: Intelligent gemstone analysis and investment insights (Preview).
- **Interactive Search**: Advanced Navbar search overlay for effortless collection browsing.
- **Secure Wishlist**: Persistent saved collection with intuitive management.
- **Concierge Integration**: Floating WhatsApp concierge for private inquiries.

## 🛠️ Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS, Framer Motion
- **Icons**: Lucide React
- **CMS**: Contentful
- **State Management**: React Hooks

## 🚀 Getting Started

### 1. Prerequisites
- Node.js 18+
- Contentful Account

### 2. Environment Setup
Create a `.env.local` file in the root and add your credentials:
```bash
# Contentful
CONTENTFUL_SPACE_ID=your_space_id
CONTENTFUL_ACCESS_TOKEN=your_access_token

# Auth
JWT_SECRET=your_jwt_secret
```

### 3. Installation
```bash
npm install
```

### 4. Run Development Server
```bash
npm run dev
```

## 📂 Project Structure
- `/app`: Next.js App Router (Pages & API)
- `/components`: Reusable UI components
- `/lib`: Utilities (Contentful client, shared data)
- `/public`: High-fidelity assets and logos
